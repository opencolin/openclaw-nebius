Getting Started

Deploy your first gateway in three steps

1

Get a Token Factory API key

Sign up at tokenfactory.nebius.com and create an API key. This key lets your gateway call LLM inference on Nebius GPUs — no need to run your own model.

  • Token Factory provides access to Llama, DeepSeek, Qwen, and more
  • Keys start with v1. — you'll paste it during deployment
  • GPU endpoints run models locally and don't need a key
Get API Key
2

Deploy an endpoint

Go to Quick Start and work through the steps: pick your gateway image, choose a model, select a region, and choose a platform. For GPU workloads no API key is required. Hit Deploy — the endpoint spins up in about a minute.

  • GPU — dedicated H200 VM, model runs locally, no API key needed
  • CPU Only — serverless, fastest cold start, uses Token Factory
  • Custom — choose exact vCPUs, RAM, and GPU model
3

Connect to your gateway

Open Endpoints to see your running endpoints. Each active endpoint has two ways to connect:

  • Terminal — opens a live SSH shell directly in the browser. Use it to inspect logs, run commands, or debug your gateway.
  • Dashboard — opens the gateway's web UI in a new tab (proxied through this server). Use it to interact with your gateway, test prompts, and monitor activity.

Endpoints that show inactive can be started from the action menu on the right.
